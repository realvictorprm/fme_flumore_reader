/* The FLUMORE FME Importing Plugin V1.0a*/
[English]

Installation steps:
1. All folders here must be copied to the FME root folder (the folder where the fme.exe is located!).
2. Ready

Usage:
1. Open the reader dialog
2. Select under additional formats the "FLUMORE" formats
4. Select a FLUMORE compatible file
5. Enable single merged feature type
6. Ready to go

[German]

Installation:
1. Alle Ordner hier müssen komplett in das Verzeichnis von FME kopiert werden (Dort wo die fme.exe lokalisiert ist!).
2. Fertig

Benutzung:
1. Reader Dialog öffnen
2. Unter weitere Formate das Format "FLUMORE" auswählen.
3. Eine Datei auswählen die mit FLUMORE kompatibel ist.
4. Die Option "Single-merged-feature-type" auswählen bzw. nur einen einzigen Feature Typ aktivieren.
5. Nun kann die Datei ausgelesen werden.
